The language is the framework.

A modern open-source scripting language for web apps and APIs.
One file. Zero boilerplate. Run with mx run app.mx.

Install Read the docs

Founded and developed by Jassim Alkharafi.

A real JSON API in five lines.

server { port: 8080 }

get /users/:id {
  let id = num(request.params.id)
  return json({ id: id, name: "Jassim" })
}

No framework imports. No middleware setup. No package.json. mx run app.mx and you have a JSON API on localhost:8080.

Batteries included.

HTTP framework

Routes, middleware, static files, cookies, request body parsing, CORS, gzip, rate limiting, graceful shutdown — all first-class language constructs.

Real-time

Server-sent events sse /events and WebSockets ws /chat as native route types. RFC 6455 in pure stdlib.

SQLite

sql.open / exec / query — proper SQL persistence built in. Pure Go driver, no CGo.

Concurrency

spawn { ... } for goroutines, channels for coordination. Modern async without callbacks or promises.

Auth

JWT, signed cookies, OAuth2 helpers for Google / GitHub / Discord / Microsoft / LinkedIn — write a login flow in 20 lines.

AI built in

ai.complete(prompt) — OpenAI or Anthropic Claude. Add an LLM endpoint to your app in one line.

Tooling

REPL, test runner, watch mode, formatter, pretty error messages with stack traces, GoReleaser binaries on every tag.

Single binary

One Go binary with one runtime dep (SQLite). No virtual machine. No node_modules. No GC tuning.

Install

From source (Go 1.25+)

$ git clone https://github.com/jlkdevelop/mxscript.git
$ cd mxscript
$ go build -o mx .
$ ./mx version

Pre-built binaries

macOS, Linux, and Windows binaries on every tagged release: github.com/jlkdevelop/mxscript/releases.

Hello world

$ mx run --eval 'print("Hello, " + uuid())'
